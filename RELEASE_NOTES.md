# Release Notes

## v0.1.0

**Highlights**
- Timed drawing sessions using a local image library
- Single‑image quick sessions (double‑click)
- Session history with calendar navigation
- Light/Dark mode

**Install**
1. Download `DrawSesh.zip` from GitHub Releases.
2. Unzip and drag `DrawSesh.app` to `Applications` (optional but recommended).
3. First launch: Right‑click the app → **Open** → **Open**.

**Notes**
This build is unsigned/notarized. macOS will show a warning the first time you open it.
